About me

My name is Henry Salliss, I am from near London in the UK! I have recently been learning front end web development, I am new to software engineering and eager to learn more and gain the knowledge I need to create slick code while ensuring the best quality and service to whatever project I am working on. I just started learning to code about 12 months ago, but I have created multiple websites and mini projects showcasing multiple different skills

Skills

I can write clean HTML markup which shows all the content needed for the web page while also keeping it semantically organised.

I can write CSS3 with a range of skills going from simple hover effects and colours, to positioning elements well on the page using Flexbox.

JavaScript is my first real programming language that I have been learning extensively for the last 6 months. I have incorporated some basic JavaScript on this website but have also delved into deeper JavaScript which takes user input, stores and then manipulates that data based on events happening in the web app. I plan on learning multiple JavaScript frameworks.
