Recipe app

The most complex project yet, using APIs calls the user can search for recipes. The user can bookmark the recipe where it is stored in local storage. The user can also create their own recipes which are immediately bookmarked. This was all created through MVC (model, view, controller) architecture.

Number Game

A number game created with JavaScript logic that supports two players taking turns on rolling a dice which stores a score which increases on every roll except when a 1 is rolled, the other player can then play. If a player chooses to hold their current score will be added to their main score. First to 100 wins.
