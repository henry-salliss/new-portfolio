Number Game

A number game created with JavaScript logic that supports two players taking turns on rolling a dice which stores a score which increases on every roll except when a 1 is rolled, the other player can then play. If a player chooses to hold their current score will be added to their main score. First to 100 wins.

Guess the number

Simple JavaScript game where a random number is generated and user has to guess, appropriate hints displayed depending on user input.

Election map

JavaScript map that is best viewed on desktop, when mouse hovers over states the winner of the election in that state is revealed, vote count changes dynamically in bottom right.

Magic 8 ball

Magic 8 ball which uses JavaScript to respond to user input to display answer to a yes or no question thought of by the user.

JavaScript clock

Clock made using JavaScript that displays different messages based on the time of day, the drop down boxes for the various events also display other images if set time is equal to the current time.
